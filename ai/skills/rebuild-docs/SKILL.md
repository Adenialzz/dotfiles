# Rebuild Docs

用户打算重写本 repo。为此，需要先完整理解现在它有什么功能。

## 内容

你需要读原来的文档和源码来整理以下内容：

- 名词、动词、引擎、点火钥匙

- 主要的模块和 call graph

- 重要的前 10 个文件/函数，及其详细功能实现

## 要求

- 将重写参考文档整理在 `rebuild-docs` 目录下

- 适当使用 ascii art 图和表格帮助说明。

- 不要动任何其他文件

- 所有文档都用中文。
